STORIES
ASCENT UNIVERSE — CHRONOLOGICAL INDEX
All fiction set in the Ascent Universe, ordered by in-universe date. Stories range from deep Dyn prehistory (~100,000 BCE) to the Apathy War (~3200 CE).
WHERE TO START ▸
Best entry point: The Facilitator — a standalone near-future AI story that seeds the entire timeline.
For the full universe: Threshold (novel) — first contact, alien archaeology, and the battle that changes everything.
For the Dyn: Seeker — short, atmospheric introduction to the alien civilisation and its deep past.
Chronological: Start with Frameshift and read top-to-bottom — the list below is already in timeline order.
Authorship note: Stories marked human were written entirely by the author. Stories marked AI-assisted were written collaboratively with Claude (Anthropic) — typically human-plotted with AI drafting and expansion. Stories marked AI-edited were human-written originals with AI-assisted revision. See the README for full details.
